A comprehensive Identity Provider Administration System.

## Features
- **User Management**: Create, update, delete users; assign roles.
- **Impersonation**: Login as users for support.
- **System Settings**: Configure password policies, audit retention, and branding.
- **Audit Logging**: Comprehensive tracking of administrative actions.

## Quick Start
1.  Run the backend (Web.IdP).
2.  Run the frontend (ClientApp) or use the hosted Razor views.
3.  Login as Admin.

# 隱私權政策（簡要）

本文件為專案的隱私權政策範本，說明我們如何處理使用者資料與 Cookie。

## 1. 資料收集
我們會蒐集必要的帳號資訊（如電子郵件、顯示名稱）以提供認證與授權服務。

## 2. Cookie 與類似技術
我們使用 Cookie 來維持使用者登入狀態、記錄使用者偏好設定，以及進行必要的安全檢查。某些分析或第三方整合（例如外部登入）可能會設定額外的 Cookie。

## 3. Cookie 同意
若需遵守法律（例如 GDPR），應顯示 Cookie 同意橫幅，並尊重使用者選擇。此專案在前端提供簡易 Cookie 同意機制，可選擇接受或拒絕非必要 cookie。

## 4. 資料保存與刪除
使用者可按照系統提供的流程請求刪除其個人資料，系統會在合理時間內處理。

## 5. 聯絡方式
如有隱私相關問題，請聯絡系統管理員或開發團隊。

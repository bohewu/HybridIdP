# 隱私權政策（簡要）

本文件為專案的隱私權政策範本，說明我們如何處理使用者資料與 Cookie。

## 1. 資料收集
我們會蒐集必要的帳號資訊（如電子郵件、顯示名稱）以提供認證與授權服務。

## 2. Cookie 與類似技術
系統使用必要的 Cookie 以維持登入狀態、記錄使用者語言與顯示偏好，以及做安全性檢查。本系統目前**不使用第三方分析服務（例如 Google Analytics）**，因此不會在預設情況下啟用分析 Cookie。

## 3. Cookie 同意
為遵守部分地區的隱私法規（例如 GDPR），系統會在使用者首次造訪時顯示簡易 Cookie 同意提示。使用者可以接受或拒絕「非必要」Cookie；系統僅在使用者同意時才會啟用那些非必要功能。

## 4. 資料保存與刪除
使用者可按照系統提供的流程請求刪除其個人資料，系統會在合理時間內處理。

## 5. 聯絡方式
如有隱私相關問題，請聯絡系統管理員或貴公司的法務／合規窗口。
